# Call Scenario Between PSTN and Mobile Station in a Mobile Satellite Wireless Network

## Project Description
This project focuses on designing and simulating a Next Generation Network (NGN) using Cisco Packet Tracer. The network integrates multiple layers and technologies to enable communication between a PSTN phone and a mobile station through a satellite wireless network.

## Network Architecture
The network is divided into several layers:
- Access Layer (end-user devices)
- Aggregation Layer
- Core Layer
- Service Layer

## My Role
I was responsible for:
- IP addressing and subnetting

- Connectivity verification
- Ensuring end-to-end communication
